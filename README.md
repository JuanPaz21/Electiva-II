# Electiva-II
El proyecto automatiza la captura y análisis de datos del mercado laboral usando RPA y herramientas de inteligencia empresarial. Con UiPath, se realiza scraping en múltiples portales de empleo para extraer información clave de las ofertas (puesto, empresa, ubicación, requisitos, salario, etc.). Los datos obtenidos se consolidan en un archivo Excel que sirve como fuente única y estructurada.

Posteriormente, la información almacenada se visualiza en Power BI, creando dashboards interactivos que permiten explorar tendencias de demanda, distribución salarial y competencias más solicitadas.

Finalmente, mediante árboles de decisión entrenados con el historial recolectado, el modelo predice el salario recomendado para un candidato en función de las habilidades y parámetros que se ingresen (años de experiencia, certificaciones, nivel de estudio, tecnología dominante, etc.). Así, la solución brinda una visión integral del mercado y una herramienta práctica para estimar la remuneración adecuada.
